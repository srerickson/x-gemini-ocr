# Document Processing with Gemini API

## Tools

- This a python project managed with `uv`
- Use `uv run python ...` to run python scripts.
- Use `uv add ...` to add additional dependencies
- use `poppler-utils` for PDF processing (in `split_pdf.sh`).

## Organization

- `Columbia-FB.pdf` is the source document. 
- `split_pdf.sh` is a shell script to split the source document into separate files per page.
- `pages/` has per-page PDFs and JPGs.
